# Habit Planner

A full-stack Habit Tracking application built with the MERN stack (MongoDB, Express, React Native/Expo, Node.js).

## Features

- **Habit Management**: Create daily, weekly, monthly, and custom habits.
- **Statistics**: Visualize progress with weekly and monthly charts.
- **Delayed Completion**: Celebrate completions with confetti before they vanish from the daily list.
- **Modern UI**: Spotlight background effect and dark mode theme.

## Tech Stack

- **Frontend**: React Native (Expo), Victory Native (charts), Reanimated.
- **Backend**: Node.js, Express, MongoDB.

## Getting Started

### Backend

Currently, backend runs on port 5000.

```bash
cd habit-planner-backend
npm install
npm run dev
```

### Mobile App

```bash
cd habit-planner-mobile
npm install
npx expo start
```
