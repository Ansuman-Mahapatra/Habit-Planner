# Habit Planner Mobile

## Setup

1.  Run `npm install`.
2.  Ensure your backend is running.
3.  Update `services/api.ts` with your backend URL.
    - If using Android Emulator, use `http://10.0.2.2:5000`
    - If using physical device, use your computer's IP `http://192.168.x.x:5000`
4.  Run `npm run android` or `npm run ios`.

## Features

- **Auth**: Login & Register
- **Dashboard**: Progress Ring, Stats, Today's Habits
- **Habits**: Create, View, Delete, Toggle Completion
- **Stats**: Weekly Bar Chart
- **Theme**: Dark Mode optimized
